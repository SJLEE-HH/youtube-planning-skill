# youtube-planning-skill

> Claude Code 스킬 — 유튜브 신규 프로그램 기획안을 완성본으로 자동 작성

주제와 라이벌 채널 정보를 주면 **프로그램명 후보·영상 구조·예상 조회수·진행자 추천·파일럿 에피소드 4개**를 한 번에 만들어줍니다.

---

## 설치

```bash
npx skills add https://github.com/SJLEE-HH/youtube-planning-skill
```

## 사용법

Claude Code에서 아래처럼 말하면 스킬이 자동으로 작동합니다.

```
[주제]로 유튜브 기획안 만들어줘.
라이벌 채널은 [채널명1], [채널명2]이고 평균 조회수는 [조회수] 정도야.
타깃은 [타깃 시청자]이고, [원하는 방향/콘셉트]로 하고 싶어.
```

**트리거 키워드**: "기획안 만들어줘", "유튜브 기획안", "프로그램 기획", "파일럿 에피소드 짜줘", "유튜브 콘텐츠 기획", "채널 기획", "새 프로그램 기획", "유튜브 시리즈 만들어줘"

### 예시

```
보험 상품 소개 유튜브 시리즈 기획안 만들어줘.
라이벌은 '삼성화재 유튜브', '메리츠화재 TV'이고 평균 조회수 5만~10만이야.
타깃은 30대 직장인이고, 어렵지 않게 일상 언어로 설명하는 채널로 만들고 싶어.
```

> 정보가 부족해도 괜찮습니다. 주제만 있으면 나머지는 스킬이 추론해서 채워줍니다.

---

## 출력 항목

| # | 섹션 | 내용 |
|---|---|---|
| 1 | 프로그램명 후보 | 5개 이상, 각각 선정 이유 포함 |
| 2 | 영상 구조 | 타임라인 형태 + 파트별 이유 |
| 3 | 예상 조회수 | 보수·중립·낙관 시나리오 + 핵심 변수 |
| 4 | 진행자·게스트 추천 | 실존 인물, ⚠️ URL 미검증 표시 포함 |
| 5 | 파일럿 에피소드 4개 | 제목·핵심 내용·기대 포인트·예상 댓글 반응 |

완성 후 자가 검증(라이벌 차별점·URL 경고·소재 중복 점검)을 자동으로 수행합니다.
