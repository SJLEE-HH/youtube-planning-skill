# YouTube 기획안 스킬 사용 가이드

## 이 스킬이 하는 일

주제, 라이벌 채널, 라이벌 조회수를 알려주면 유튜브 프로그램 기획안을 완성형으로 만들어준다.

출력 항목:
1. 프로그램명 후보 리스트 (5개 이상, 각각 이유 포함)
2. 영상 구조 / 스크립트 구조 (타임라인 형태)
3. 예상 조회수 (보수/중립/낙관 시나리오 + 핵심 변수)
4. 진행자 후보 및 게스트 추천
5. 파일럿 에피소드 4개 (제목 + 내용 + 예상 댓글 반응)

---

## 사용 방법

Claude Code를 열고 아래처럼 말하면 끝이다. 스킬이 자동으로 작동한다.

### 기본 입력 형식

```
[주제]로 유튜브 기획안 만들어줘.
라이벌 채널은 [채널명1], [채널명2]이고 평균 조회수는 [조회수] 정도야.
타깃은 [타깃 시청자]이고, [원하는 방향/콘셉트]로 하고 싶어.
```

### 실제 예시

```
보험 상품 소개 유튜브 시리즈 기획안 만들어줘.
라이벌은 '삼성화재 유튜브', '메리츠화재 TV'이고 평균 조회수 5만~10만이야.
타깃은 30대 직장인이고, 어렵지 않게 일상 언어로 설명하는 채널로 만들고 싶어.
```

```
LIFEPLUS TV 건강 콘텐츠 기획안 짜줘.
라이벌은 '삼성생명 유튜브', '한화생명 공식채널'이야.
40~50대 자산관리 관심층 타깃이고, 건강검진 결과 해석 같은 실용 정보 위주로.
```

---

## 정보가 없을 때

라이벌 조회수, 타깃, 방향을 몰라도 괜찮다.
주제와 라이벌 채널명만 줘도 나머지는 스킬이 추론해서 채워준다.

```
ESG 주제로 유튜브 기획안 만들어줘. 라이벌은 '한화그룹 공식채널'이야.
```

## 스킬 파일 위치

스킬 본체: `~/.claude/skills/youtube-planning/SKILL.md`

수정이 필요하면 이 파일을 직접 편집하거나 Claude Code에 수정 요청하면 된다.
